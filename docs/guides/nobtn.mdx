# 不想总是弹出翻译按钮

默认情况下，划词翻译会在你划选文本后弹出翻译按钮。如果你不希望弹出翻译按钮，那么你可以进入[设置页](options.mdx) >【网页划词】>【划词设置】，勾选【禁用网页划词】。这样一来，划词后就不会弹出翻译按钮了，但你仍然可以通过辅助键、快捷键和右键菜单等方式翻译你选中的文本。

你还可以打开[扩展弹出页](popup.mdx)，将网站加入黑名单。加入黑名单的网站不会弹出翻译按钮，详情请查看[扩展弹出页](popup.mdx)内的说明。
